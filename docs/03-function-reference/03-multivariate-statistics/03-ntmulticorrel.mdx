# NTMULTICORREL

Returns mean vector, standard deviation vector and correlation matrix

## Syntax

```excel
NTMULTICORREL(
    data_set,
    calc_mean,
    calc_stdev
)
```

### Parameters

- **_Data set_** is a set of series of variables.
- **_Calc mean_** is a logical value that determines means of each variables are computed or not.
- **_Calc stdev_** is a logical value that determines standard deviations of each variables are computed or not.
