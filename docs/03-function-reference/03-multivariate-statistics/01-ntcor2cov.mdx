# NT2COR2COV

Converts standard deviation vector and correlation matrix to covariance matrix

## Syntax

```excel
NTCOR2COV(
    stdev_vector,
    correlation_matrix
)
```

### Parameters

- **_Stdev vector_** is a 1-dim vector whose components are standard deviations of each variable.
- **_Correlation matrix_** is a correlation matrix.
