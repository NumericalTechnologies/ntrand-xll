# NTNORMDIST

Returns the standard normal distribution for the specified parameters.

## Syntax

```excel
NTNORMDIST(
    x,
    Cumulative
)
```

### Parameters

- **_x_** is the value for which you want the distribution $(x>0)$.
- **_Cumulative_** is a logical value that determines the form of the function. If cumulative is TRUE, NTNORMDIST returns the cumulative distribution function; if FALSE, it returns the probability mass function.

## Remarks

- When argument Cumulative=TRUE, this function returns same result as Excel function "NORMSDIST" does.
- When argument Cumulative=FALSE, this function returns same result as Excel function "NORMSDIST(x,0,1,FALSE)" does.
