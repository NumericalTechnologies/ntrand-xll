# NTWEIBULLDIST

Returns the Weibull distribution for the specified parameters.

## Syntax

```excel
NTWEIBULLDIST(
    x,
    Alpha,
    Beta,
    Cumulative
)
```

### Parameters

- **_x_** is the value for which you want the distribution.
- **_Alpha_** is a parameter of the distribution $\alpha(>0)$.
- **_Beta_** is a parameter of the distribution $\beta(>0)$.
- **_Cumulative_** is a logical value that determines the form of the function. If cumulative is TRUE, NTBETADIST returns the cumulative distribution function; if FALSE, it returns the probability density function.

## Remarks

- This function is same as Excel function "WEIBULL".
