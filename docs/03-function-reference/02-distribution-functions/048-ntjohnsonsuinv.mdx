# NTJOHNSONSUINV

Returns the inverse of the cumulative distribution function for a specified Johnson SU distribution.

## Syntax

```excel
NTJOHNSONSUINV(
    prob,
    Gamma,
    Delta,
    Lambda,
    Xi
)
```

### Parameters

- **_prob_** is a probability corresponding to the normal distribution $(0<prob<1)$.
- **_Gamma_** is a parameter of the distribution $\gamma$.
- **_Delta_** is a parameter of the distribution $\delta(>0)$.
- **_Lambda_** is a parameter of the distribution $\lambda(>0)$.
- **_Xi_** is a parameter of the distribution $\xi$.
