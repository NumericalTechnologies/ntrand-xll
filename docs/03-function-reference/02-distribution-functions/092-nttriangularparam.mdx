# NTTRIANGULARPARAM

Returns parameters of the triangular distribution for the specified mean, standard deviation and skewness.

## Syntax

```excel
NTTRIANGULARPARAM(
    Mean,
    Stdev,
    Skew
)
```

### Parameters

- **_Mean_** is the arithmetic mean of the distribution.
- **_Stdev_** is the standard deviation of the distribution (Positive value).
- **_Skew_** is the skewness of the distribution.
