# NTPOISSONDIST

Returns the Poisson distribution for the specified parameters.

## Syntax

```excel
NTPOISSONDIST(
    k,
    Mean,
    Cumulative
)
```

### Parameters

- **_k_** is the value for which you want the distribution $k(\geq 0)$ (Integer).
- **_Mean_** is a parameter of the distribution $\nu$.
- **_Cumulative_** is a logical value that determines the form of the function. If cumulative is TRUE, NTPOISSONDIST returns the cumulative distribution function; if FALSE, it returns the probability mass function.

## Remarks

- This function is same as Excel function "POISSON".
