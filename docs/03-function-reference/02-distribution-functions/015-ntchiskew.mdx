# NTCHISKEW

Returns [skewness](/docs/glossary#skewness) of a specified chi distribution.

## Syntax

```excel
NTCHISKEW(
    N
)
```

### Parameters

- **_N_** is a parameter of the distribution $N$ (Positive integer).

## Remarks

The [skewness](/docs/glossary#skewness) of the distribution is given as:

$$
\frac{\mu(3)-3\mu(2)\mu(1)+2\mu^3(1)}{\sigma^3}
$$

$$
\mu(r)=\frac{2^{\frac{r}{2}}\Gamma\left(\frac{N+r}{2}\right)}{\Gamma\left(\frac{N}{2}\right)}
$$

, where $\sigma$ is [standard deviation](/docs/glossary#standard-deviation) and $\Gamma(\cdot)$ is [gamma function](/docs/glossary#gamma-function).
