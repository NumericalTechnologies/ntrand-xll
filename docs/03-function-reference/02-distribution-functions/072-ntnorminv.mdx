# NTNORMINV

Returns the inverse of the cumulative distribution function for a specified standard normal distribution.

## Syntax

```excel
NTNORMINV(
    prob
)
```

### Parameters

- **_prob_** is a probability corresponding to the normal distribution $(0<prob<1)$.

## Remarks

- This function is same as Excel function "NORMSINV".
