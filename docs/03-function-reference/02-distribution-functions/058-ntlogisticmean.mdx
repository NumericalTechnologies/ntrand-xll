# NTLOGISTICMEAN

Returns mean of a specified logistic distribution.

## Syntax

```excel
NTLOGISTICMEAN(
    M
)
```

### Parameters

- **_M_** is a parameter of the distribution $m$.

## Remarks

- This function returns same value as argument M
