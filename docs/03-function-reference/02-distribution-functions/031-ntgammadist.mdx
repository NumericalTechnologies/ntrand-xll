# NTGAMMADIST

Returns the gamma distribution for the specified parameters.

## Syntax

```excel
NTGAMMADIST(
    x,
    Alpha,
    Beta,
    Cumulative
)
```

### Parameters

- **_x_** is the value for which you want the distribution ($x\geq 0$).
- **_Alpha_** is a parameter of the distribution $\alpha(>0)$.
- **_Beta_** is a parameter of the distribution $\beta(>0)$.
- **_Cumulative_** is a logical value that determines the form of the function. If cumulative is TRUE, NTGAMMADIST returns the cumulative distribution function; if FALSE, it returns the probability density function.

## Remarks

- [Probability density function](/docs/glossary#probability-density-function) is given as:

$$
  f(x)=\frac{1}{\beta^{\alpha}\Gamma(\alpha)}\exp\left(-\frac{x}{\beta}\right)x^{\alpha-1}
$$

, where $\Gamma(\cdot)$ is [gamma function](/docs/glossary#gamma-function).

- [Cumulative distribution function](/docs/glossary#cumulative-distribution-function) is given as:

  $$
  F(x)=\frac{\Gamma_{\frac{x}{\beta}}(\alpha)}{\Gamma(\alpha)}
  $$

  , where $\Gamma_{x}(\cdot)$ is [incomplete gamma function](/docs/glossary#incomplete-gamma-function).
