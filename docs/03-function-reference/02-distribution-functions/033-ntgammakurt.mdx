# NTGAMMAKURT

Returns [kurtosis](/docs/glossary#kurtosis) of a specified gamma distribution.

## Syntax

```excel
NTGAMMAKURT(
    Alpha
)
```

### Parameters

- **_Alpha_** is a parameter of the distribution $\alpha(>0)$.

## Remarks

The [kurtosis](/docs/glossary#kurtosis) of the distribution is given as:

$$
\frac{6}{\alpha}
$$
