# NTJOHNSONSUPARAM

Returns parameters of the Johnson SU distribution for the specified mean standard deviation, skewness and kurtosis.

## Syntax

```excel
NTJOHNSONSUPARAM(
    Mean,
    Stdev,
    Skew,
    Kurt
)
```

### Parameters

- **_Mean_** is the arithmetic mean of the distribution.
- **_Stdev_** is the standard deviation of the distribution (Positive value).
- **_Skew_** is the skewness of the distribution.
- **_Kurt_** is the kurtosis of the distribution (Positive value).
