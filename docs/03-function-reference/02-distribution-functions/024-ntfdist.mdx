# NTCHIDIST

Returns the f distribution for the specified parameters.

## Syntax

```excel
NTFDIST(
    x,
    N1,
    N2,
    Cumulative
)
```

### Parameters

- **_x_** is the value for which you want the distribution ($x\geq 0$).
- **_N1_** is a parameter of the distribution $N_1$ (Positive integer).
- **_N2_** is a parameter of the distribution $N_2$ (Positive integer).
- **_Cumulative_** is a logical value that determines the form of the function. If cumulative is TRUE, NTFDIST returns the cumulative distribution function; if FALSE, it returns the probability density function.

## Remarks

- [Probability density function](/docs/glossary#probability-density-function) is given as:

$$
  f(x)=\frac{\Gamma\left(\frac{N_1+N_2}{2}\right)\left(\frac{N_1}{N_2}\right)^\frac{N_1}{2}x^{\frac{N_1}{2}-1}}{\Gamma\left(\frac{N_1}{2}\right)\Gamma\left(\frac{N_2}{2}\right)\left(1+\frac{N_1}{N_2}x\right)^{\frac{N_1+N_2}{2}}}
$$

, where $\Gamma(\cdot)$ is [gamma function](/docs/glossary#gamma-function).

- [Cumulative distribution function](/docs/glossary#cumulative-distribution-function) is given as:

  $$
  F(x)=I_\gamma\left(\frac{N_1}{2},\frac{N_2}{2}\right)
  $$

  , where $\gamma=\frac{N_1x}{N_2+N_1x}$ and $I_{x}(\cdot,\cdot)$ is [regularized incomplete beta function](/docs/glossary#regularized-incomplete-beta-function).
