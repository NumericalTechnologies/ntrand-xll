# NTCHIMEAN

Returns [mean](/docs/glossary#mean) of a specified chi distribution.

## Syntax

```excel
NTCHIMEAN(
    N
)
```

### Parameters

- **_N_** is a parameter of the distribution $N$ (Positive integer).

## Remarks

The [mean](/docs/glossary#mean) of the distribution is given as:

$$
\sqrt{2}\frac{\Gamma\left(\frac{N+1}{2}\right)}{\Gamma\left(\frac{N}{2}\right)}
$$

, where $\Gamma(\cdot)$ is [gamma function](/docs/glossary#gamma-function).
