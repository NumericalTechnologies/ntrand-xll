# NTLOGNORMDIST

Returns the log normal distribution for the specified parameters.

## Syntax

```excel
NTLOGNORMDIST(
    x,
    M,
    S,
    Cumulative
)
```

### Parameters

- **_x_** is the value for which you want the distribution $(x>0)$.
- **_M_** is a parameter of the distribution $M$.
- **_S_** is a parameter of the distribution $S(>0)$.
- **_Cumulative_** is a logical value that determines the form of the function. If cumulative is TRUE, NTLOGNORMDIST returns the cumulative distribution function; if FALSE, it returns the probability mass function.
