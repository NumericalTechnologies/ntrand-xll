# NTCHISQDIST

Returns the chi square distribution for the specified parameters.

## Syntax

```excel
NTCHISQDIST(
    x,
    N,
    Cumulative
)
```

### Parameters

- **_x_** is the value for which you want the distribution ($x\geq 0$).
- **_N_** is a parameter of the distribution $N$ (Positive integer).
- **_Cumulative_** is a logical value that determines the form of the function. If cumulative is TRUE, NTCHISQDIST returns the cumulative distribution function; if FALSE, it returns the probability density function.

## Remarks

- [Probability density function](/docs/glossary#probability-density-function) is given as:

$$
  f(x)=\frac{1}{2^{\frac{N}{2}}\Gamma\left(\frac{N}{2}\right)}\exp\left(-\frac{x}{2}\right)x^{\frac{N}{2}-1}
$$

, where $\Gamma(\cdot)$ is [gamma function](/docs/glossary#gamma-function).

- [Cumulative distribution function](/docs/glossary#cumulative-distribution-function) is given as:

$$
  F(x)=\frac{\Gamma_{\frac{x}{2}}\left(\frac{N}{2}\right)}{\Gamma\left(\frac{N}{2}\right)}
$$

, where $\Gamma_{x}(\cdot)$ is [incomplete gamma function](/docs/glossary#incomplete-gamma-function).
