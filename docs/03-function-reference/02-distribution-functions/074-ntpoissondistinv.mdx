# NTPOISSONDISTINV

Returns the inverse of the cumulative distribution function for a specified Poisson distribution.

## Syntax

```excel
NTPOISSONDISTINV(
    prob,
    Mean
)
```

### Parameters

- **_prob_** is a probability corresponding to the normal distribution $(0<prob<1)$.
- **_Mean_** is a parameter of the distribution.
