# NTFMEAN

Returns [mean](/docs/glossary#mean) of a specified f distribution.

## Syntax

```excel
NTFMEAN(
    N2
)
```

### Parameters

- **_N2_** is a parameter of the distribution $N_2(>2)$.

## Remarks

The [mean](/docs/glossary#mean) of the distribution is given as:

$$
\frac{N_2}{N_2-2}
$$
