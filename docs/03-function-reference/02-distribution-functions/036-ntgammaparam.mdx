# NTGAMMAPARAM

Returns parameters of the gamma distribution for the specified mean and standard deviation.

## Syntax

```excel
NTGAMMAPARAM(
    Mean,
    Stdev
)
```

### Parameters

- **_Mean_** is the arithmetic mean of the distribution.
- **_Stdev_** is the standard deviation of the distribution (Positive value).

## Remarks

Parameters are estimated as:

$$
\alpha=\frac{m^2}{\sigma^2},\quad \beta=\frac{\sigma^2}{m}
$$

where $m$ is [mean](/docs/glossary#mean) and $\sigma$ is [standard deviation](/docs/glossary#standard-deviation).
