# NTPOISSONMOM

Returns mean, standard deviation, skewness and kurtosis of a specified Poisson distribution.

## Syntax

```excel
NTPOISSONMOM(
    Mean
)
```

### Parameters

- **_Mean_** is the arithmetic mean of the distribution.
