# NTWEIBULLPARAM

Returns parameters of the Weibull distribution for the specified mean and standard deviation.

## Syntax

```excel
NTWEIBULLPARAM(
    Mean,
    Stdev
)
```

### Parameters

- **_Mean_** is the arithmetic mean of the distribution.
- **_Stdev_** is the standard deviation of the distribution (Positive value).
