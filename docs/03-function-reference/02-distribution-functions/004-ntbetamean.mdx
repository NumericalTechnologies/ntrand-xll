# NTBETAMEAN

Returns [standard deviation](/docs/glossary#standard-deviation) of a specified beta distribution.

## Syntax

```excel
NTBETAMEAN(
    Alpha,
    Beta
)
```

### Parameters

- **_Alpha_** is a parameter of the distribution $\alpha(>0)$.
- **_Beta_** is a parameter of the distribution $\beta(>0)$.

## Remarks

The [mean](/docs/glossary#mean) of the distribution is given as:

$$
\frac{\alpha}{\alpha+\beta}
$$
