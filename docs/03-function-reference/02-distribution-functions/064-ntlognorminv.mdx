# NTLOGNORMINV

Returns the inverse of the cumulative distribution function for a specified log normal distribution.

## Syntax

```excel
NTLOGNORMINV(
    prob,
    M,
    S
)
```

### Parameters

- **_prob_** is a probability corresponding to the normal distribution $(0<prob<1)$.
- **_M_** is a parameter of the distribution $M$.
- **_S_** is a parameter of the distribution $S(>0)$.
