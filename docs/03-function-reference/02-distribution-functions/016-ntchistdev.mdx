# NTCHISTDEV

Returns [standard deviation](/docs/glossary#standard-deviation) of a specified chi distribution.

## Syntax

```excel
NTCHISTDEV(
    N
)
```

#### Parameters

- **_N_** is a parameter of the distribution $N$ (Positive integer).

## Remarks

The [standard deviation](/docs/glossary#standard-deviation) of the distribution is given as:

$$
\left(N-2\left[\frac{\Gamma\left(\frac{N+1}{2}\right)}{\Gamma\left(\frac{N}{2}\right)}\right]^2\right)^{1/2}
$$
