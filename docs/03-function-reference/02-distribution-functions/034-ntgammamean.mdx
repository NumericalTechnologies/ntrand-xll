# NTGAMMAMEAN

Returns [mean](/docs/glossary#mean) of a specified gamma distribution.

## Syntax

```excel
NTGAMMAMEAN(
    Alpha,
    Beta
)
```

### Parameters

- **_Alpha_** is a parameter of the distribution $\alpha(>0)$.
- **_Beta_** is a parameter of the distribution $\beta(>0)$ of the distribution is given as:

$$
\alpha\beta
$$
