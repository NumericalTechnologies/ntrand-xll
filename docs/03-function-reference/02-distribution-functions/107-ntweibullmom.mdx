# NTWEIBULLMOM

Returns mean, standard deviation, skewness and kurtosis of a specified Weibull distribution.

## Syntax

```excel
NTWEIBULLMOM(
    Alpha,
    Beta
)
```

### Parameters

- **_Alpha_** is a parameter of the distribution $\alpha(>0)$.
- **_Beta_** is a parameter of the distribution $\beta(>0)$.
