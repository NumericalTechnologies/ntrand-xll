# NTTDISTINV

Returns the inverse of the cumulative distribution function for a specified Student's t distribution distribution.

## Syntax

```excel
NTTDISTINV(
    prob,
    N
)
```

### Parameters

- **_prob_** is a probability corresponding to the normal distribution $(0<prob<1)$.
- **_N_** is a parameter of the distribution (Positive integer).
