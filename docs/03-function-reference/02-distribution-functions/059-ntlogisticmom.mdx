# NTLOGISTICMOM

Returns mean, standard deviation, skewness and kurtosis of a specified logistic distribution.

## Syntax

```excel
NTLOGISTICMOM(
    M,
    B
)
```

### Parameters

- **_M_** is a parameter of the distribution $m$.
- **_B_** is a parameter of the distribution $b(>0)$.
