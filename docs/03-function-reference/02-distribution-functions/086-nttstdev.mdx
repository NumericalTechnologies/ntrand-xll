# NTTSTDEV

Returns [standard deviation](/docs/glossary#standard-deviation) of a specified Student's t distribution.

## Syntax

```excel
NTTSTDEV(
    N
)
```

### Parameters

- **_N_** is a parameter of the distribution $N(>2)$.

## Remarks

The [standard deviation](/docs/glossary#standard-deviation) of the distribution is given as:

$$
\sqrt{\frac{N}{N-2}}
$$
