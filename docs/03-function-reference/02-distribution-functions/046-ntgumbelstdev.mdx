# NTGUMBELSTDEV

Returns standard deviation of a specified Gumbel distribution.

## Syntax

```excel
NTGUMBELSTDEV(
    Beta
)
```

### Parameters

- **_Beta_** is a parameter of the distribution $\beta(>0)$.

## Remarks

The [standard deviation](/docs/glossary#standard-deviation) of the distribution is given as:

$$
\frac{\beta\pi}{\sqrt{6}}
$$
