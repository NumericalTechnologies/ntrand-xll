# NTGUMBELMEAN

Returns mean of a specified Gumbel distribution.

## Syntax

```excel
NTGUMBELMEAN(
    Alpha,
    Beta
)
```

### Parameters

- **_Alpha_** is a parameter of the distribution $\alpha$.
- **_Beta_** is a parameter of the distribution $\beta(>0)$.

## Remarks

The [mean](/docs/glossary#mean) of the distribution is given as:

$$
\alpha+\gamma\beta=\alpha+0.57721566490153\beta
$$

, where $\gamma$ is [Euler's constant](/docs/glossary#euler-8217-s-constant).
