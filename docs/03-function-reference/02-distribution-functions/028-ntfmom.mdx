# NTFMOM

Returns mean, standard deviation, skewness and kurtosis of a specified F distribution.

## Syntax

```excel
NTFMOM(
    N1,
    N2
)
```

### Parameters

- **_N1_** is a parameter of the distribution $N_1$ (Positive integer).
- **_N2_** is a parameter of the distribution $N_2(>2)$.

## Example

- The example may be easier to understand if you copy it to a blank worksheet
