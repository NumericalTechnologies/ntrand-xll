# NTBETAMOM

Returns mean, standard deviation, skewness and kurtosis of a specified beta distribution.

## Syntax

```excel
NTBETAMOM(
    Alpha,
    Beta
)
```

### Parameters

- **_Alpha_** is a parameter of the distribution $\alpha(>0)$.
- **_Beta_** is a parameter of the distribution $\beta(>0)$.
