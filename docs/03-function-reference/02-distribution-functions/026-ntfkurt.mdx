# NTFKURT

Returns [kurtosis](/docs/glossary#kurtosis) of a specified F distribution.

## Syntax

```excel
NTFKURT(
    N1,
    N2
)
```

### Parameters

- **_N1_** is a parameter of the distribution $N_1$ (Positive integer).
- **_N2_** is a parameter of the distribution $N_2(>8)$.

## Remarks

The [kurtosis](/docs/glossary#kurtosis) of the distribution is given as:

$$
\frac{12[(N_2-2)^2(N_2-4)+N_1(N_1+N_2-2)(5N_2-22)]}{N_1(N_2-6)(N_2-8)(N_1+N_2-2)}
$$
