# NTGUMBELDIST

Returns the Gumbel distribution for the specified parameters.

## Syntax

```excel
NTGUMBELDIST(
    x,
    Alpha,
    Beta,
    Cumulative
)
```

### Parameters

- **_x_** is the value for which you want the distribution.
- **_Alpha_** is a parameter of the distribution $\alpha$.
- **_Beta_** is a parameter of the distribution $\beta(>0)$.
- **_Cumulative_** is a logical value that determines the form of the function. If cumulative is TRUE, NTGUMBELDIST returns the cumulative distribution function; if FALSE, it returns the probability mass function.
