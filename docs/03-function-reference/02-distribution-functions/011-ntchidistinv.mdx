# NTCHIDISTINV

Returns the inverse of the cumulative distribution function for a specified Chi distribution.

## Syntax

```excel
NTCHIDISTINV(
    prob,
    N
)
```

### Parameters

- **_prob_** is a probability corresponding to the normal distribution $(0<prob<1)$.
- **_N_** is a parameter of the distribution (Positive integer).
