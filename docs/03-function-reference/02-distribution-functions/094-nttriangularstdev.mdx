# NTTRIANGULARSTDEV

Returns standard deviation of a specified triangular distribution.

## Syntax

```excel
NTTRIANGULARSTDEV(
    Min,
    Max,
    Mode
)
```

### Parameters

- **_Min_** is a lower limit of a support of the distribution $a$.
- **_Max_** is a upper limit of a support of the distribution $b$.
- **_Mode_** is a mode of the distribution $c$.

## Remarks

- Restrictions of arguments:

$$
  a<c<b
$$
