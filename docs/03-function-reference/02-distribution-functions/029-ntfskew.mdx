# NTFSKEW

Returns [skewness](/docs/glossary#skewness) of a specified F distribution.

## Syntax

```excel
NTFSKEW(
    N1,
    N2
)
```

### Parameters

- **_N1_** is a parameter of the distribution $N_1$ (Positive integer).
- **_N2_** is a parameter of the distribution $N_2(>6)$.

## Remarks

The [skewness](/docs/glossary#skewness) of the distribution is given as:

$$
\frac{(2N_1+N_2-2)\sqrt{8(N_2-4)}}{\sqrt{N_1(N_1+N_2-2)}(N_2-6)}
$$
