# NTGUMBELPARAM

Returns parameters of the Gumbel distribution for the specified mean and standard deviation.

## Syntax

```excel
NTGUMBELPARAM(
    Mean,
    Stdev
)
```

### Parameters

- **_Mean_** is the arithmetic mean of the distribution.
- **_Stdev_** is the standard deviation of the distribution (Positive value).
