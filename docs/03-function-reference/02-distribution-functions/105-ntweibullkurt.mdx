# NTWEIBULLKURT

Returns kurtosis of a specified Weibull distribution.

## Syntax

```excel
NTWEIBULLKURT(
    Alpha,
    Beta
)
```

### Parameters

- **_Alpha_** is a parameter of the distribution $\alpha(>0)$.
- **_Beta_** is a parameter of the distribution $\beta(>0)$.
