# NTLOGNORMMOM

Returns mean, standard deviation, skewness and kurtosis of a specified log normal distribution.

## Syntax

```excel
NTLOGNORMMOM(
    M,
    S
)
```

### Parameters

- **_M_** is a parameter of the distribution $M$.
- **_S_** is a parameter of the distribution $S(>0)$.
