# NTGAMMADISTINV

Returns the inverse of the cumulative distribution function for a specified gamma distribution.

## Syntax

```excel
NTGAMMADISTINV(
    prob,
    Alpha,
    Beta
)
```

### Parameters

- **_prob_** is a probability corresponding to the normal distribution $(0<prob<1)$.
- **_Alpha_** is a parameter of the distribution $\alpha(>0)$.
- **_Beta_** is a parameter of the distribution $\beta(>0)$.
