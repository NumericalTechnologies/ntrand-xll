# NTTRIANGULARINV

Returns the inverse of the cumulative distribution function for a specified triangular distribution.

## Syntax

```excel
NTTRIANGULARINV(
    prob,
    Min,
    Max,
    Mode
)
```

### Parameters

- **_prob_** is a probability corresponding to the normal distribution.
- **_Min_** is a lower limit of a support of the distribution $a$.
- **_Max_** is a upper limit of a support of the distribution $b$.
- **_Mode_** is a mode of the distribution $c$.

## Remarks

- Restrictions of arguments:

$$
  a<c<b
$$
