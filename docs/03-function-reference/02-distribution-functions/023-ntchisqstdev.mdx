# NTCHISQSTDEV

Returns [standard deviation](https://www.ntrand.com/glossary/#local_standard_deviation) of a specified chi distribution.

## Syntax

```excel
NTCHISQSTDEV(
    N
)
```

### Parameters

- **_N_** is a parameter of the distribution $N$ (Positive integer).

## Remarks

The [standard deviation](https://www.ntrand.com/glossary/#local_standard_deviation) of the distribution is given as:

$$
\sqrt{2N}
$$
