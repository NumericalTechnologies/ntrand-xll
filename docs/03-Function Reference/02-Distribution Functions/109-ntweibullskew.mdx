# NTWEIBULLSKEW

Returns skewness of a specified Weibull distribution.

## Syntax

```excel
NTWEIBULLSKEW(
    Alpha,
    Beta
)
```

### Parameters

- **_Alpha_** is a parameter of the distribution $\alpha(>0)$.
- **_Beta_** is a parameter of the distribution $\beta(>0)$.
