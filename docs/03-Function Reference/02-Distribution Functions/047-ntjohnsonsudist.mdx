# NTJOHNSONSUDIST

Returns the Johnson SU distribution for the specified parameters.

## Syntax

```excel
NTJOHNSONSUDIST(
    x,
    Gamma,
    Delta,
    Lambda,
    Xi,
    Cumulative
)
```

### Parameters

- **_x_** is the value for which you want the distribution.
- **_Gamma_** is a parameter of the distribution $\gamma$.
- **_Delta_** is a parameter of the distribution $\delta(>0)$.
- **_Lambda_** is a parameter of the distribution $\lambda(>0)$.
- **_Xi_** is a parameter of the distribution $\xi$.
- **_Cumulative_** is a logical value that determines the form of the function. If cumulative is TRUE, NTJOHNSONSUDIST returns the cumulative distribution function; if FALSE, it returns the probability mass function.
