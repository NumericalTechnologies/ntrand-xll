# NTGAMMAMOM

Returns mean, standard deviation, skewness and kurtosis of a specified gamma distribution.

## Syntax

```excel
NTGAMMAMOM(
    Alpha,
    Beta
)
```

### Parameters

- **_Alpha_** is a parameter of the distribution $\alpha(>0)$.
- **_Beta_** is a parameter of the distribution $\beta(>0)$.
