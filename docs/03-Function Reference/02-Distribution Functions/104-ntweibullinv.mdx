# NTWEIBULLINV

Returns the inverse of the cumulative distribution function for a specified Weibull distribution.

## Syntax

```excel
NTWEIBULLINV(
    prob,
    Alpha,
    Beta
)
```

### Parameters

- **_prob_** is a probability corresponding to the normal distribution.
- **_Alpha_** is a parameter of the distribution $\alpha(>0)$.
- **_Beta_** is a parameter of the distribution $\beta(>0)$.

## Remarks

The return value x for probability P is

$$
x=\alpha\left(-\log P\right)^{1/\beta}
$$
