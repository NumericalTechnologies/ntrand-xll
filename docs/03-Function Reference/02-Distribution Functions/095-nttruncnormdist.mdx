# NTTRUNCNORMDIST

Returns the truncated normal distribution for the specified parameters.

## Syntax

```excel
NTTRUNCNORMDIST(
    x,
    Min,
    Max,
    Mu,
    Sigma,
    Cumulative
)
```

### Parameters

- **_x_** is the value for which you want the distribution.
- **_Min_** is a lower limit of a support of the distribution $a$.
- **_Max_** is a upper limit of a support of the distribution $b$.
- **_Mu_** is a parameter the distribution $\mu$.
- **_Sigma_** is a parameter the distribution $\sigma(>0)$.
- **_Cumulative_** is a logical value that determines the form of the function. If cumulative is TRUE, NORMDIST returns the cumulative distribution function; if FALSE, it returns the probability mass function.

## Remarks

- Restrictions of arguments:

$$
  a<b
$$

$$
  a\leq x\leq b
$$
