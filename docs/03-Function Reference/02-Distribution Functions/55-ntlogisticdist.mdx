# NTLOGISTICDIST

Returns the logistic distribution for the specified parameters.

## Syntax

```excel
NTLOGISTICDIST(
    x,
    M,
    B,
    Cumulative
)
```

### Parameters

- **_x_** is the value for which you want the distribution.
- **_M_** is a parameter of the distribution $m$.
- **_B_** is a parameter of the distribution $b(>0)$.
- **_Cumulative_** is a logical value that determines the form of the function. If cumulative is TRUE, NTLOGISTICDIST returns the cumulative distribution function; if FALSE, it returns the probability mass function.
