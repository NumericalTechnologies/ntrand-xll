# NTTKURT

Returns [kurtosis](https://www.ntrand.com/glossary/#local_kurtosis) of a specified Student's t distribution.

## Syntax

```excel
NTTKURT(
    N
)
```

### Parameters

- **_N_** is a parameter of the distribution $N(>4)$.

## Remarks

The [kurtosis](https://www.ntrand.com/glossary/#local_kurtosis) of the distribution is given as:

$$
\frac{6}{N-4}
$$
