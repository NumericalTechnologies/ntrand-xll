# NTCHIDIST

Returns the chi distribution for the specified parameters.

## Syntax

```excel
NTCHIDIST(
    x,
    N,
    Cumulative
)
```

### Parameters

- **_x_** is the value for which you want the distribution ($x\geq 0$).
- **_N_** is a parameter of the distribution (Positive integer).
- **_Cumulative_** is a logical value that determines the form of the function. If cumulative is TRUE, NTCHIDIST returns the cumulative distribution function; if FALSE, it returns the probability density function.

## Remarks

- [Probability density function](https://www.ntrand.com/glossary/#local_probability) is given as:

$$
  f(x)=\frac{1}{2^{\frac{N}{2}-1}\Gamma\left(\frac{N}{2}\right)}\exp\left(-\frac{x^2}{2}\right)x^{N-1}
$$

, where $\Gamma(\cdot)$ is [gamma function](https://www.ntrand.com/glossary/#local_Gamma).

- [Cumulative distribution function](https://www.ntrand.com/glossary/#local_cumulative) is given as:

  $$
    F(x)=\frac{\Gamma_{\frac{x^2}{2}}\left(\frac{N}{2}\right)}{\Gamma\left(\frac{N}{2}\right)}
  $$

  , where $\Gamma_{x}(\cdot)$ is [incomplete gamma function](https://www.ntrand.com/glossary/#local_incomplete_gamma).
