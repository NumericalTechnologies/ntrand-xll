# NTLOGISTICINV

Returns the inverse of the cumulative distribution function for a specified logistic distribution.

## Syntax

```excel
NTLOGISTICINV(
    prob,
    M,
    B
)
```

### Parameters

- **_prob_** is a probability corresponding to the normal distribution $(0<prob<1)$.
- **_M_** is a parameter of the distribution $m$.
- **_B_** is a parameter of the distribution $b(>0)$.
