# NTCHISQKURT

Returns [kurtosis](https://www.ntrand.com/glossary/#local_kurtosis) of a specified chi square distribution.

## Syntax

```excel
NTCHISQKURT(
    N
)
```

### Parameters

- **_N_** is a parameter of the distribution $N$ (Positive integer).

## Remarks

The [kurtosis](https://www.ntrand.com/glossary/#local_kurtosis) of the distribution is given as:

$$
\frac{12}{N}
$$
