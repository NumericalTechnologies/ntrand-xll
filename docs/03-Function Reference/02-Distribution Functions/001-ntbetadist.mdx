# NTBETADIST

Returns the beta distribution for the specified parameters.

## Syntax

```excel
NTBETADIST(
    x,
    Alpha,
    Beta,
    Cumulative
)
```

### Parameters

- **_x_** is the value for which you want the distribution ($0\leq x\leq 1$).
- **_Alpha_** is a parameter of the distribution $\alpha(>0)$.
- **_Beta_** is a parameter of the distribution $\beta(>0)$.
- **_Cumulative_** is a logical value that determines the form of the function. If cumulative is TRUE, NTBETADIST returns the cumulative distribution function; if FALSE, it returns the probability density function.

## Remarks

- [Probability density function](https://www.ntrand.com/glossary/#local_probability) is given as:

$$
  f(x)=\frac{x^{\alpha-1}(1-x)^{\beta-1}}{B(\alpha,\beta)}
$$

, where $B(\cdot,;\cdot)$ is [beta function](https://www.ntrand.com/glossary/#local_Beta).

- [Cumulative distribution function](https://www.ntrand.com/glossary/#local_cumulative) is given as:

$$
  F(x)=I_{x}(\alpha,;\beta)
$$

, where $I_x(\cdot,;\cdot)$ is [regualarized incomplete beta function](https://www.ntrand.com/glossary/#local_Regularized_incomplete_beta).

- When the argument Cumulative=TRUE, this function returns same result as Excel function "BETADIST" does.
