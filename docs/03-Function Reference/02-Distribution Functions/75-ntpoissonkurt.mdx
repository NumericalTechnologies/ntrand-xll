# NTPOISSONKURT

Returns kurtosis of a specified Poisson distribution.

## Syntax

```excel
NTPOISSONKURT(
    Mean
)
```

### Parameters

- **_Mean_** is the arithmetic mean of the distribution (Positive value).

## Remarks

The [kurtosis](https://www.ntrand.com/glossary/#local_kurtosis) of the distribution is given as:

$$
\frac{1}{\textit{Mean}}
$$
