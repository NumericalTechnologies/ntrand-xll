# NTBETAKURT

Returns [kurtosis](https://www.ntrand.com/glossary/#local_kurtosis) of a specified beta distribution.

## Syntax

```excel
NTBETAKURT(
    Alpha,
    Beta
)
```

### Parameters

- **_Alpha_** is a parameter of the distribution $\alpha(>0)$.
- **_Beta_** is a parameter of the distribution $\beta(>0)$.

## Remarks

The [kurtosis](https://www.ntrand.com/glossary/#local_kurtosis) of the distribution is given as:

$$
6\frac{\alpha^3-\alpha^2(2\beta-1)+\beta^2(\beta+1)-2\alpha\beta(\beta+2)}{\alpha\beta(\alpha+\beta+2)(\alpha+\beta+3)}
$$
