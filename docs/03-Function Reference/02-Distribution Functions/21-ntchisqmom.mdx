# NTCHISQMOM

Returns mean, standard deviation, skewness and kurtosis of a specified chisq distribution.

## Syntax

```excel
NTCHISQMOM(
    N
)
```

### Parameters

- **_N_** is a parameter of the distribution $N$ (Positive integer).

## Example
