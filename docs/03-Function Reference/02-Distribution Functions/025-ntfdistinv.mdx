# NTFDISTINV

Returns the inverse of the cumulative distribution function for a specified F distribution.

## Syntax

```excel
NTFDISTINV(
    prob,
    N1,
    N2
)
```

### Parameters

- **_prob_** is a probability corresponding to the normal distribution $(0<prob<1)$.
- **_N1_** is a parameter of the distribution $N_1$ (Positive integer).
- **_N2_** is a parameter of the distribution $N_2$ (Positive integer).
