# NTCOV2COR

Converts covariance matrix to standard deviation vector and correlation matrix

## Syntax

```excel
NTCOV2COR(
    covariance_matrix,
    calc_stdev_vector
)
```

### Parameters

- **_Covariance matrix_** is a covariance matrix.
- **_Calc stdev vector_** is a logical value that determines standard deviations of each variables are computed or not.
