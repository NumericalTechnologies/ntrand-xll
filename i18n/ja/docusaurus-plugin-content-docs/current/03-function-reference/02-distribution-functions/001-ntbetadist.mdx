# NTBETADIST

ベータ分布の確率の値を返します。

## 書式

```excel
NTBETADIST(
    x,
    Alpha,
    Beta,
    Cumulative
)
```

#### 引数

- **_x_** 関数に代入する値を指定します（$0\leq x\leq 1$）。
- **_Alpha_** 分布のパラメータ $\alpha(>0)$ を指定します。
- **_Beta_** 分布のパラメータ $\beta(>0)$ を指定します。
- **_Cumulative_** 計算に使用する指数関数の形式を論理値で指定します。*Cumulative* に TRUE を指定すると累積分布関数の値が計算され、FALSE を指定すると確率密度関数の値が計算されます。

## 解説

- [確率密度関数](https://www.ntrand.com/jp/glossary/#local_probability)は次式で与えられます :

  $$
  f(x)=\frac{x^{\alpha-1}(1-x)^{\beta-1}}{B(\alpha,\beta)}
  $$

  ここで $B(\cdot,\cdot)$ は[ベータ関数](https://www.ntrand.com/jp/glossary/#local_Beta)です。

- [累積分布関数](https://www.ntrand.com/jp/glossary/#local_cumulative)は次式で与えられます :

  $$
  F(x)=I_{x}(\alpha,\beta)
  $$

  ここで $I_{x}(\cdot,\cdot)$ は [正規化された不完全ベータ関数](https://www.ntrand.com/jp/glossary/#local_Regularized_incomplete_beta)です。

- 引数 *Cumulative* が TRUE の場合、この関数は Excel 関数 "BETADIST" と同等です。
