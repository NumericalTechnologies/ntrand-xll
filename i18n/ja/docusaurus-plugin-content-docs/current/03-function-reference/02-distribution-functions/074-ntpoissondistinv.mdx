# NTPOISSONDISTINV

ポアソン分布の累積分布関数の逆関数の値を返します。

## 書式

```excel
NTPOISSONDISTINV(
    prob,
    Mean
)
```

### 引数

- **_prob_** ポアソン分布における確率を指定します $(0<prob<1)$
- **_Mean_** 分布のパラメータ $\nu$ を指定します。

## 使用例

- 使用例を新規のワークシートにコピーすると、計算結果を確認できます。
