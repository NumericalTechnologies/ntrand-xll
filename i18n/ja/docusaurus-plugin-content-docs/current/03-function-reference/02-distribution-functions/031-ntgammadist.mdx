# NTGAMMADIST

[ガンマ分布](https://www.ntrand.com/jp/gamma-distribution/)布の確率の値を返します。

## 書式

```excel
NTGAMMADIST(
    x,
    Alpha,
    Beta,
    Cumulative
)
```

### 引数

- **_x_** 関数に代入する値を指定します（$x\geq 0$）。
- **_Alpha_** 分布のパラメータ $\alpha$ を指定します。
- **_Beta_** 分布のパラメータ $\beta(>0)$ を指定します。
- **_Cumulative_** 計算に使用する指数関数の形式を論理値で指定します。*Cumulative* に TRUE を指定すると累積分布関数の値が計算され、FALSE を指定すると確率密度関数の値が計算されます。

## 解説

- [確率密度関数](https://www.ntrand.com/jp/glossary/#local_probability)

  $$
  f(x)=\frac{1}{\beta^{\alpha}\Gamma(\alpha)}\exp\left(-\frac{x}{\beta}\right)x^{\alpha-1}
  $$

  ここで $\Gamma(\cdot)$ は [ガンマ関数](https://www.ntrand.com/glossary/#local_Gamma)です。

- [累積分布関数](https://www.ntrand.com/jp/glossary/#local_cumulative)

  $$
  F(x)=\frac{\Gamma_{\frac{x}{\beta}}(\alpha)}{\Gamma(\alpha)}
  $$

  ここで $\Gamma_{x}(\cdot)$ は [不完全ガンマ関数](https://www.ntrand.com/glossary/#local_incomplete_gamma)です。

- 引数 *Cumulative* が TRUE の場合、この関数は Excel 関数 "GAMMADIST" と同等です。

2.  ヘルプ トピックにある使用例を選択します（行番号および列番号を除く）。
