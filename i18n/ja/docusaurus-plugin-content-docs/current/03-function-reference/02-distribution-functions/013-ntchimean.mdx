# NTCHIMEAN

[カイ分布](https://www.ntrand.com/jp/chi-distribution/)の[平均](https://www.ntrand.com/jp/glossary/#local_mean)を返します。

## 書式

```excel
NTCHIMEAN(
    N
)
```

### 引数

- **_N_** 分布のパラメータ $N$ を指定します（正の整数）。

## 解説

分布の[平均](https://www.ntrand.com/jp/glossary/#local_mean) は次式で与えられます。

$$
\sqrt{2}\frac{\Gamma\left(\frac{N+1}{2}\right)}{\Gamma\left(\frac{N}{2}\right)}
$$

ここで $\Gamma(\cdot)$ は [ガンマ関数](https://www.ntrand.com/glossary/#local_Gamma)です。

## 使用例

- 使用例を新規のワークシートにコピーすると、計算結果を確認できます。
