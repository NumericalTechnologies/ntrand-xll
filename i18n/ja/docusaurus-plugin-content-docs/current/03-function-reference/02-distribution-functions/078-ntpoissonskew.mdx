# NTPOISSONSKEW

ポアソン分布の歪度を返します。

## 書式

```excel
NTPOISSONSKEW(
    Mean
)
```

### 引数

- **_Mean_** 分布の[平均](https://www.ntrand.com/jp/glossary/#local_mean)を指定します。

## 解説

分布の[歪度](https://www.ntrand.com/jp/glossary/#local_skewness)は次式で与えられます。

$$
\frac{1}{\sqrt{\textit{Mean}}}
$$

## 使用例

- 使用例を新規のワークシートにコピーすると、計算結果を確認できます。
