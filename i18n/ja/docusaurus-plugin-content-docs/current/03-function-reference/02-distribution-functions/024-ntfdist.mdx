# NTFDIST

[F 分布](../../05-gallery-of-distributions/f-distribution.mdx)布の確率の値を返します。

## 書式

```excel
NTFDIST(
    x,
    N1,
    N2,
    Cumulative
)
```

### 引数

- **_x_** 関数に代入する値を指定します（$x\geq 0$）。
- **_N1_** 分布のパラメータ $N_1$ を指定します（正の整数）。
- **_N2_** 分布のパラメータ $N_2$ を指定します（正の整数）。
- **_Cumulative_** 計算に使用する指数関数の形式を論理値で指定します。*Cumulative* に TRUE を指定すると累積分布関数の値が計算され、FALSE を指定すると確率密度関数の値が計算されます。

## 解説

- [確率密度関数](/docs/glossary#probability-density-function)

  $$
  f(x)=\frac{\Gamma\left(\frac{N_1+N_2}{2}\right)\left(\frac{N_1}{N_2}\right)^\frac{N_1}{2}x^{\frac{N_1}{2}-1}}{\Gamma\left(\frac{N_1}{2}\right)\Gamma\left(\frac{N_2}{2}\right)\left(1+\frac{N_1}{N_2}x\right)^{\frac{N_1+N_2}{2}}}
  $$

  ここで $\Gamma(\cdot)$ は [ガンマ関数](/docs/glossary#gamma-function)です。

- [累積分布関数](/docs/glossary#cumulative-distribution-function)

  $$
  F(x)=I_\gamma\left(\frac{N_1}{2},\frac{N_2}{2}\right)
  $$

  ここで $\gamma=\frac{N_1x}{N_2+N_1x}$ で $I_{x}(\cdot,\cdot)$ は [正規化された不完全ベータ関数](/docs/glossary#regularized-incomplete-beta-function)です。

2.  ヘルプ トピックにある使用例を選択します（行番号および列番号を除く）。

    ![Selecting an example from Help](./../../assets/functions/HowToCopy.png)

3.  Ctrl キーを押しながら C キーを押します。

5.  計算結果と結果を返す数式の表示を切り替えるには、Ctrl キーを押しながら ` (アクサン グラーブ) キーを押すか、または [ツール] メニューの [ワークシート分析] をポイントし、[ワークシート分析モード] をクリックします。
