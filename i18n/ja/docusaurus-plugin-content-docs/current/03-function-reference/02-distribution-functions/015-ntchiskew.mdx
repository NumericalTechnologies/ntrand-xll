# NTCHISKEW

[カイ分布](https://www.ntrand.com/jp/chi-distribution/)の[歪度](https://www.ntrand.com/jp/glossary/#local_skewness)を返します。

## 書式

```excel
NTCHISKEW(
    N
)
```

### 引数

- **_N_** 分布のパラメータ $N$ を指定します（正の整数）。

## 解説

分布の[歪度](https://www.ntrand.com/jp/glossary/#local_skewness)は次式で与えられます。

$$
\frac{\mu(3)-3\mu(2)\mu(1)+2\mu^3(1)}{\sigma^3}
$$

$$
\mu(r)=\frac{2^{\frac{r}{2}}\Gamma\left(\frac{N+r}{2}\right)}{\Gamma\left(\frac{N}{2}\right)}
$$

ここで $\sigma$ は [標準偏差](https://www.ntrand.com/glossary/#local_standard_deviation) で ![\Gamma(\cdot)](https://s0.wp.com/latex.php?latex=%5CGamma%28%5Ccdot%29&bg=T&fg=000000&s=0 "\\Gamma(\\cdot)") は [ガンマ関数](https://www.ntrand.com/glossary/#local_Gamma)です。
