# NTTRUNCNORMDIST

切断正規分布の確率の値を返します。

## 書式

```excel
NTTRUNCNORMDIST(
    x,
    Min,
    Max,
    Mu,
    Sigma,
    Cumulative
)
```

### 引数

- **_x_** 関数に代入する値を指定します。
- **_Min_** 分布の下限 $a$ を指定します。
- **_Max_** 分布の上限 $b$ を指定します。
- **_Mu_** 分布のパラメータ $\mu$ を指定します。
- **_Sigma_** 分布のパラメータ $\sigma(>0)$ を指定します。
- **_Cumulative_** 計算に使用する指数関数の形式を論理値で指定します。*Cumulative* に TRUE を指定すると累積分布関数の値が計算され、FALSE を指定すると確率密度関数の値が計算されます。

## 解説

- パラメータは $a<b$ を満たす必要があります。

- 引数 *x* の範囲は $a\leq x\leq b$ です。
