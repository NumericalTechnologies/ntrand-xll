# NTPOISSONDIST

ポアソン分布の確率の値を返します。

## 書式

```excel
NTPOISSONDIST(
    k,
    Mean,
    Cumulative
)
```

### 引数

- **_k_** 関数に代入する値を指定します $k(\geq 0)$。
- **_Mean_** 分布のパラメータ $\nu$ を指定します。
- **_Cumulative_** 計算に使用する指数関数の形式を論理値で指定します。*Cumulative* に TRUE を指定すると累積分布関数の値が計算され、FALSE を指定すると確率質量関数の値が計算されます。

## 解説

- この関数は Excel関数 "[POISSON](http://office.microsoft.com/ja-jp/excel/HP052092161041.aspx)" と同等です。

## 使用例

- 使用例を新規のワークシートにコピーすると、計算結果を確認できます。
