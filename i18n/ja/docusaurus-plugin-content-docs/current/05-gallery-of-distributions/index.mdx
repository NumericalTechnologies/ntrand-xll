# 確率分布の世界

### 確率分布の世界へようこそ！

二の足を踏んでないで興味のあるところをのぞいてみてください！興味ない？いやいや、以外に面白い事実があるんですよ...例えば[正規分布](https://www.ntrand.com/jp/normal-distribution-single/ "正規分布")とか。\
将来的に**コツコツ**増やしていく予定です、実は。
